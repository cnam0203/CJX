Documentation - Import

1. Import a touchpoint:

Following these steps:

  • Step 1: Click Database button on the left column
  • Step 2: Click Touchpoint button
  • Step 3: Click Add New Item on the top right region
  • Step 4: Input and Click Save button
  • Optional : With fields that suggest multiple choices (Action_Type, Device_OS,... ), user can provide more choices by Add New Item for those fields in corresponding tables

2. Import multiple touchpoints (CSV File):

Following these steps:

  • Step 1: Click Import button on the left column
  • Step 2: Click Create mapping file button
  • Step 3: Input names of matching columns and their converting functions
  • Step 4: Click Import CSV File button
  • Step 5: Click Upload CSV File button and pick one CSV file
  • Step 6: Select Mapping Type: defined mapping file or manually mapping (manipulate)
  • Step 7: If user select "Manipulate",must match each column in CSV to each field in Data Model
  • Step 8: Click Import button and wait until finishing
