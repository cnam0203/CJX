API Integration

Customer Journey Master enables businesses to integrate their customer touchpoint data collected from websites, e-commerce sites, or mobile sales applications through Restful API.

Step 1: Create Secret Key

  • Click Database button on the left column
  • Click SecretKey button
  • Click Add New Item button
  • Input a secret key and Save

Step 2: Prepare Restful API

  • End Point: http://${domain_website}||${ip_address}/journey/import/add-new-touchpoint
  • Method : 'POST'
  • Header : add field 'Authorization: Bearer ${SECRET_KEY}'
  • Parameters: list of fields in Touchpoint Model
  • * If not exist one of these parameters ('time', 'channel_type', 'device_browser', 'device_os', 'device_category', 'geo_continent', 'geo_country', 'geo_city'), CJM will detect and fill in
  • Example: http://127.0.0.0/journey/import/add-new-touchpoint?customer_id=1&action_type=login&channel_type=website

Step 3: Wait for Response

  • A 200 response (OK) indicates success
  • A 400 response indicates an error in parameters given
